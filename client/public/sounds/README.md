# Notification Sounds

This folder contains notification sound files for the KEEPSAKE application.

## Available Sounds

Place the following sound files in this directory:

- `notification-default.mp3` - Default notification sound
- `notification-chime.mp3` - Chime sound
- `notification-bell.mp3` - Bell sound
- `notification-ding.mp3` - Ding sound
- `notification-ping.mp3` - Ping sound
- `notification-pop.mp3` - Pop sound

## Sound File Requirements

- Format: MP3 (recommended) or WAV
- Duration: 0.5-2 seconds (short and pleasant)
- File size: Under 100KB for optimal performance
- Sample rate: 44.1kHz or 48kHz

## Free Sound Resources

You can find free notification sounds from:
- https://notificationsounds.com/
- https://freesound.org/
- https://mixkit.co/free-sound-effects/notification/
- https://soundbible.com/

## Custom Sounds

Users can also provide their own custom sound URLs in the notification settings.
